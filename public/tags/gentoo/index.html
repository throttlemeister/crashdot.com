Gentoo on Windows with WSL2

Introduction WSL 2 on Windows 10 introduced the ability to run a native linux kernel on your computer while using Windows 10 as your main operating system. Instead of emulating a linux kernel, like WSL 1 does, WSL 2 uses a hypervisor to run linux in parallel with Windows. To be able to run WSL 2 on Windows 10, installation of Windows 10 feature update 2004 or newer is required.
